## Sections
- **Home** – Introduction and a quick overview of who I am
- **About** – Full resume including work experience, education, certifications, and skills
- **Technologies** – Tools and languages I work with including Python, JavaScript, React, Bootstrap, and Node.js
- **Projects** – Showcase of my projects including Job Tracker, Web Scraper, and Email Automation
- **Contact** – Email and social links
- **Now** – What I'm currently working on and learning

## Technologies Used
- HTML5
- CSS3
- Bootstrap 5
- Vanilla JavaScript

## Features
- Responsive design that works on mobile and desktop
- Dark mode toggle with localStorage persistence
- Smooth scrolling navigation
- Active navigation link highlighting on scroll

## Projects Featured

- **Web Scraper** – Streamlit dashboard scraping government job listings from DPSA
- **Email Automation** – Python tool that sends automatic follow-up emails for job applications
